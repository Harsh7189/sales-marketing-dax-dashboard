# HR Analytics Dashboard (Power BI)
Project Overview

This project focuses on building an interactive HR Analytics dashboard using Power BI to analyze workforce metrics and support data-driven HR decisions. The dashboard provides a clear view of employee trends, attrition patterns, and workforce distribution across departments and demographics.

# Objectives

Analyze employee count and attrition trends

Track recruitment funnel performance

Understand department-wise workforce distribution

Evaluate gender diversity across the organization

# Dashboard Features

Employee headcount and attrition KPIs

Attrition trend analysis over time

Department-wise and gender-wise workforce distribution

Interactive slicers for department, gender, and time period

Clean, user-friendly layout for easy interpretation

# Key Learnings

Data modeling and relationship building in Power BI

Writing and optimizing DAX measures for KPIs

Designing dashboards focused on business users

Converting HR data into actionable insights

# Tools & Technologies

Power BI

DAX

Excel (data preparation)

# Outcome

The dashboard enables HR teams to quickly identify workforce trends, monitor attrition, and make informed strategic decisions using interactive visual insights.
